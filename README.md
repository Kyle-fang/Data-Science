## 导论
### &emsp;数据
   - 数据无处不在：用户每次点击，网站都会记录下来；我们每时每刻的位置和速度，都会被智能手机记录下来
   - 数据的范畴比数字要大得多，如**文字，图片，视频**都是数据
   - 数据本身是客观存在的，但其范畴是随着文明的进程不断变化和扩大的（在计算机出现之前，一般书籍上的文字内容是不被看成是数据的）
   - 数据中隐藏的信息和知识是客观存在的，但是只有具有相关领域专业知识的人才能将它们挖掘出来
